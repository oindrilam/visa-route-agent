# VisaRoute Agent

A concierge visa route optimizer for Indian passport holders.

The agent helps users compare self-application, assisted visa services, corporate travel desks, and expert help using visa purpose, document readiness, active effort hours, calendar days, mandatory presence, and risk.

## MVP Scope

- Applicant: Indian passport holder
- Visa purposes: Tourist, Student, Business
- Output:
  - visa route recommendation
  - estimated calendar days
  - estimated active effort hours
  - interview/biometrics requirement
  - self vs assisted route recommendation

## Safety Note

This project provides planning assistance only. Users should verify final visa rules from official embassy, consulate, or government sources.
